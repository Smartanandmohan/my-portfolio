# Anand Mohan Dubey | 3D Space Theme Portfolio

A premium, highly interactive 3D portfolio website showcasing software development expertise, modern web apps, SaaS platforms, and responsive user experiences. Built with a modern tech stack featuring Next.js, React, Three.js, and Framer Motion.

## 🚀 Features

- **Interactive 3D Stars Background**: Built using Three.js and `@react-three/fiber` for an immersive, high-performance visual experience.
- **Dynamic Animations**: Smooth scroll-triggered animations and entry transitions powered by Framer Motion.
- **Fully Responsive Design**: Tailored layout supporting seamless interaction across mobile, tablet, and desktop devices.
- **Personalized Sections**:
  - **About Me**: Highlights background in software engineering and technical training programs from IIT Bombay and IIT Patna.
  - **Skills Showcase**: Categorized list of frontend, backend, and development tools with interactive badges.
  - **Project Showcase**: Display of active SaaS platforms and cloned marketplaces with live links.
  - **Services**: Detailed grid of expertise from frontend development to SaaS architecting and AI integration.
- **SEO Optimized**: Advanced SEO settings with semantic HTML structures and JSON-LD Schema integration for search engines.

## ⚙️ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Install the project dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000` to view the website.

### Production Build

To generate an optimized production build of the static site:
```bash
npm run build
```

The output will be generated inside the `.next` directory.
